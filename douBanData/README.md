# vue-cli Element 单页面demo请求豆瓣/QQ音乐榜单


## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at http://localhost:8097/module/Element.html
npm run dev

# build for production with minification
npm run build
```
